Manual

OBSなどの配信ソフトからRTMPで映像を送信し、RTSP形式で視聴者に届けるストリーミングサービスです。登録不要・無料でご利用いただけます。

📡 配信URL（RTMP）
rtmp://rtmp.vcha.jp/live/[ストリームキー]
▶ 視聴URL（RTSP）
rtsp://rtmp.vcha.jp/live/[ストリームキー]
01

02

URL仕様

ストリームキーには任意の文字列を使用できます。英数字・ハイフン・アンダースコアを推奨します。

# 配信URL（OBS等に設定）
サーバー : rtmp://rtmp.vcha.jp/live
キー    : mystream   ← 任意の文字列

# 視聴URL（VRChat / VLC等に入力）
rtsp://rtmp.vcha.jp/live/mystream

ストリームキーの衝突について

状況
動作
1人目が配信中
✓ そのまま継続
2人目が同じキーで接続
✗ 2人目の接続を拒否
⚠️短いキー（3文字以下など）は他の利用者と衝突する可能性があります。8文字以上のランダムな文字列を推奨します。
03

OBS 設定手順

配信設定

  1. OBS Studioを開き、「ファイル」→「設定」を開く
  2. 左メニューの「配信」をクリック
  3. 「サービス」を カスタム... に変更
  4. 以下の値を入力する
  5. 「OK」をクリックして保存し、「配信開始」をクリック
設定項目入力値
サーバーrtmp://rtmp.vcha.jp/live
ストリームキー任意の文字列（例：mystream）

推奨エンコード設定

⚠️帯域上限が5Mbpsのため、ビットレートは4500Kbps以下を推奨します。超過すると映像が乱れる場合があります。
設定項目推奨値
エンコーダx264 または H.264（ハードウェア）
ビットレート3000〜4500 Kbps
解像度1280×720（720p）または 1920×1080（1080p）
FPS30
キーフレーム間隔2秒
04

視聴方法

VRChat Video Player

VRChatのワールド内Video PlayerにRTSP URLを入力してください。

rtsp://rtmp.vcha.jp/live/[ストリームキー]
💡RTSPに対応したワールドでのみ利用できます。配信開始後にURLを入力してください。

VLC Media Player

  1. VLC Media Playerを開く
  2. 「メディア」→「ネットワークストリームを開く」をクリック
  3. URLを入力：rtsp://rtmp.vcha.jp/live/[ストリームキー]
  4. 「再生」をクリック
05

よくある質問

ストリームキーは事前登録が必要ですか？▸
不要です。任意の文字列をキーとして設定するだけで配信を開始できます。アカウント登録も必要ありません。
同時に何人まで視聴できますか？▸
視聴者数に制限はありません。何人でも同時に視聴可能です。
配信が繋がらない場合は？▸
  • ストリームキーが他の配信者と重複していないか確認してください
  • ビットレートが5Mbpsを超えていないか確認してください
  • ファイアウォールでポート1935（RTMP）が許可されているか確認してください
視聴URLが再生されない場合は？▸
  • 配信が開始されているか確認してください（配信開始前は視聴できません）
  • ストリームキーのスペルが正しいか確認してください
  • 使用するプレイヤーがRTSPに対応しているか確認してください
録画はできますか？▸
本サービス側での録画機能はありません。OBS等のソフトウェアで手元に録画してください。
利用規約・禁止事項は？▸
  • 著作権を侵害するコンテンツの配信
  • 他者への誹謗中傷・ハラスメントを含むコンテンツ
  • わいせつ・暴力的なコンテンツ
  • サービスへの意図的な負荷をかける行為
